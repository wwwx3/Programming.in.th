## Trik

1 second, 32 megabytes

![Trik cup movement](Trik.png)

สมชายไปเที่ยวงานวัดแห่งหนึ่ง ซึ่งมีการแสดงมายากลสลับลูกบอลในแก้วทึบ นักมายากลมีแก้วอยู่ 3 ใบวางเรียงกัน
โดยเริ่มต้นลูกบอลจะอยู่ในแก้วซ้ายสุด

นักมายากลจะสลับแก้วทีละสองใบอย่างรวดเร็ว โดยการสลับแต่ละครั้งถูกเข้ารหัสเป็นตัวอักษร **A**, **B**, หรือ **C**
ดังรูปด้านบน

รหัสแต่ละตัวหมายถึงการสลับตำแหน่งของแก้วดังนี้

- **A** : สลับแก้วใบที่ 1 และ 2  
- **B** : สลับแก้วใบที่ 2 และ 3  
- **C** : สลับแก้วใบที่ 1 และ 3  

โจทย์ให้เขียนโปรแกรมเพื่อรับรหัสการสลับเหล่านี้ และหาตำแหน่งสุดท้ายของลูกบอล

---

### ข้อมูลนำเข้า  

มีบรรทัดเดียว เป็นสายอักขระความยาวไม่เกิน 50 ตัวอักษร ซึ่งประกอบด้วยตัวอักษร **A**, **B**, และ **C**

---

### ข้อมูลส่งออก  

พิมพ์ตัวเลขเพียง 1 ตัว  
- พิมพ์ `1` ถ้าลูกบอลอยู่ในแก้วซ้ายสุด  
- พิมพ์ `2` ถ้าลูกบอลอยู่ในแก้วกลาง  
- พิมพ์ `3` ถ้าลูกบอลอยู่ในแก้วขวาสุด  

---

### ตัวอย่าง  

**ข้อมูลนำเข้า**
```

AB

```

**ข้อมูลส่งออก**
```

3

```

**ข้อมูลนำเข้า**
```

CBABCACCC

```

**ข้อมูลส่งออก**
```

1
