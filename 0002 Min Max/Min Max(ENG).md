## Min Max

**Time limit:** 1 second
**Memory limit:** 64 MB

Given a set of integers, find the minimum value and the maximum value, then print them.

---

### Input

- Line 1: A positive integer **n** ($1 \le n \le 1000$), the number of values  
- Lines 2 to $(n + 1)$: Each line contains an integer **Aáµ¢**, where  
  $-2{,}000{,}000{,}000 \le A_i \le 2{,}000{,}000{,}000$


---

### Output

* Line 1: An integer **m**, the minimum value in the data set
* Line 2: An integer **M**, the maximum value in the data set

---

### Example

**Input**

```
5
1
2
3
4
5
```

**Output**

```
1
5
```
