## Matrix Addition

**เวลา:** 1 วินาที
**หน่วยความจำ:** 64 MB

โจทย์ให้บวกเมทริกซ์ 2 เมทริกซ์ ขนาด ($m \times n$) ตามที่กำหนด

---

### ข้อมูลนำเข้า

* บรรทัดที่ 1: จำนวนเต็มบวก **m** และ **n** ($1 \le m, n \le 100$) แสดงมิติของเมทริกซ์
* บรรทัดที่ 2 ถึงบรรทัดที่ (m + 1): แต่ละบรรทัดมีจำนวนเต็ม **n** จำนวน แทนสมาชิกของเมทริกซ์ที่หนึ่ง
* บรรทัดที่ (m + 2) ถึงบรรทัดที่ (2m + 1): แต่ละบรรทัดมีจำนวนเต็ม **n** จำนวน แทนสมาชิกของเมทริกซ์ที่สอง

สามารถสมมติได้ว่าสมาชิกของเมทริกซ์ทุกตัวมีค่าอยู่ในช่วง
$-2{,}000{,}000{,}000 \le A_{i,j} \le 2{,}000{,}000{,}000$

---

### ข้อมูลส่งออก

มีทั้งหมด **m** บรรทัด แต่ละบรรทัดแสดงสมาชิกของเมทริกซ์ผลลัพธ์ โดยจัดรูปแบบเหมือนกับข้อมูลนำเข้า

---

### ตัวอย่าง

**ข้อมูลนำเข้า**

```
3 3
1 2 3
3 2 1
1 3 2
1 1 1
1 1 1
1 1 1
```

**ข้อมูลส่งออก**

```
2 3 4
4 3 2
2 4 3
```
