## Character Checker

**Time limit:** 1 second
**Memory limit:** 64 MB

Given a string, determine whether it contains:

* only lowercase letters,
* only uppercase letters,
* or a mix of both.

---

### Input

The input consists of one line containing a string **s** made only of English letters:

* `'a'` to `'z'` or `'A'` to `'Z'`
* length does not exceed $10{,}000$

---

### Output

Print one line according to the following rules:

* If the string contains **only uppercase** letters, print:
  `All Capital Letter`
* If the string contains **only lowercase** letters, print:
  `All Small Letter`
* If the string contains **both uppercase and lowercase** letters, print:
  `Mix`

---

### Examples

**Input**

```
ABCDEFG
```

**Output**

```
All Capital Letter
```

**Input**

```
abcdefg
```

**Output**

```
All Small Letter
```

**Input**

```
AbCdEfG
```

**Output**

```
Mix
```
