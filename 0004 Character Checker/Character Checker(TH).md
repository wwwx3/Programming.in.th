## Character Checker

**เวลา:** 1 วินาที
**หน่วยความจำ:** 64 MB

โจทย์ให้ตรวจสอบว่า “สายอักขระ” ที่กำหนดให้เป็น

* ตัวอักษรพิมพ์เล็กทั้งหมด
* ตัวอักษรพิมพ์ใหญ่ทั้งหมด
* หรือมีทั้งพิมพ์เล็กและพิมพ์ใหญ่ผสมกัน

---

### ข้อมูลนำเข้า

มีข้อมูลนำเข้าเพียง 1 บรรทัด คือสายอักขระ **s** ซึ่งประกอบด้วยตัวอักษรภาษาอังกฤษเท่านั้น

* `'a'` ถึง `'z'` หรือ `'A'` ถึง `'Z'`
* ความยาวไม่เกิน (10{,}000) ตัวอักษร

---

### ข้อมูลส่งออก

พิมพ์คำตอบ 1 บรรทัด ตามเงื่อนไขต่อไปนี้

* ถ้าสายอักขระเป็น **ตัวพิมพ์ใหญ่ทั้งหมด** ให้พิมพ์
  `All Capital Letter`
* ถ้าสายอักขระเป็น **ตัวพิมพ์เล็กทั้งหมด** ให้พิมพ์
  `All Small Letter`
* ถ้าสายอักขระมี **พิมพ์เล็กและพิมพ์ใหญ่ผสมกัน** ให้พิมพ์
  `Mix`

---

### ตัวอย่าง

**ข้อมูลนำเข้า**

```
ABCDEFG
```

**ข้อมูลส่งออก**

```
All Capital Letter
```

**ข้อมูลนำเข้า**

```
abcdefg
```

**ข้อมูลส่งออก**

```
All Small Letter
```

**ข้อมูลนำเข้า**

```
AbCdEfG
```

**ข้อมูลส่งออก**

```
Mix
```
