## ABC

**เวลา:** 1 วินาที
**หน่วยความจำ:** 32 MB

กำหนดจำนวนเต็มบวก 3 จำนวน คือ **A**, **B**, และ **C** โดยค่าทั้งสามอาจจะไม่ได้เรียงลำดับมาให้ แต่เราทราบแน่นอนว่า
[
A < B < C
]
โจทย์ให้เรียงตัวเลขทั้งสามตาม “ลำดับ” ที่กำหนดมาในบรรทัดที่สอง

---

### ข้อมูลนำเข้า

* บรรทัดที่ 1: จำนวนเต็มบวก 3 จำนวน แทนค่า **A B C** (อาจไม่เรียงลำดับ) โดยแต่ละจำนวนมีค่าไม่เกิน 100
* บรรทัดที่ 2: สายอักขระยาว 3 ตัว ประกอบด้วยตัวอักษร **A**, **B**, **C** (ไม่มีเว้นวรรค) แสดงลำดับที่ต้องการ

---

### ข้อมูลส่งออก

พิมพ์จำนวน 3 จำนวนเรียงตามลำดับที่กำหนด โดยเว้นวรรค 1 ช่องระหว่างตัวเลข

---

### ตัวอย่าง

**ข้อมูลนำเข้า**

```
1 5 3
ABC
```

**ข้อมูลส่งออก**

```
1 3 5
```

**ข้อมูลนำเข้า**

```
6 4 2
CAB
```

**ข้อมูลส่งออก**

```
6 2 4
```
