## Grading

**เวลา:** 1 วินาที
**หน่วยความจำ:** 64 MB

ในการเรียนวิชาคอมพิวเตอร์ของโรงเรียนแห่งหนึ่ง มีการเก็บคะแนนดังนี้

* คะแนนเก็บ 30 คะแนน
* คะแนนสอบกลางภาค 30 คะแนน
* คะแนนสอบปลายภาค 40 คะแนน

รวมทั้งหมด 100 คะแนน

หลังจากจบภาคการศึกษา ฝ่ายทะเบียนต้องการให้ครูกรอกคะแนนของนักเรียนลงในระบบคอมพิวเตอร์ เพื่อคำนวณเกรดของนักเรียนแต่ละคน แต่เนื่องจากยังไม่มีโปรแกรมใช้งาน จึงขอให้คุณช่วยเขียนโปรแกรมสำหรับตัดเกรดให้

---

### ข้อมูลนำเข้า

มีข้อมูลนำเข้า 3 บรรทัด

* บรรทัดที่ 1: จำนวนเต็ม **a** (0 ≤ a ≤ 30) แทนคะแนนเก็บ
* บรรทัดที่ 2: จำนวนเต็ม **b** (0 ≤ b ≤ 30) แทนคะแนนสอบกลางภาค
* บรรทัดที่ 3: จำนวนเต็ม **c** (0 ≤ c ≤ 40) แทนคะแนนสอบปลายภาค

---

### ข้อมูลส่งออก

แสดงอักขระ 1 ค่า แทนเกรดของนักเรียน โดยพิจารณาจากคะแนนรวมตามเกณฑ์ต่อไปนี้

| คะแนนรวม | เกรด |
| -------- | ---- |
| 80 – 100 | A    |
| 75 – 79  | B+   |
| 70 – 74  | B    |
| 65 – 69  | C+   |
| 60 – 64  | C    |
| 55 – 59  | D+   |
| 50 – 54  | D    |
| 0 – 49   | F    |
